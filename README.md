# Pick a Card — เว็บดูดวงความรักออนไลน์

เว็บไซต์ดูดวงความรักด้วยการเลือกไพ่ทาโรต์ 1 ใน 4 กอง

## โครงสร้างไฟล์

```
duangjai/
├── index.html          (หน้าแรก - หน้าหลักดูดวง)
├── about.html          (หน้าเกี่ยวกับเรา)
├── how-to.html         (หน้าวิธีการใช้งาน)
├── faq.html            (หน้าคำถามที่พบบ่อย)
├── css/
│   └── style.css       (สไตล์ทั้งหมด)
├── js/
│   ├── readings.js     (คำทำนาย 4 กอง - แก้คำทำนายที่นี่)
│   └── main.js         (ตรรกะการทำงานของเว็บ)
└── images/
    ├── hero.jpg        (รูปหลักหน้า Hero)
    ├── card-back.png   (หลังไพ่)
    ├── logo.png        (โลโก้)
    ├── favicon.png     (ไอคอนบน browser tab)
    └── og-image.jpg    (รูปสำหรับแชร์ social)
```

## วิธีแก้ไขในอนาคต

**แก้คำทำนาย:** เปิดไฟล์ `js/readings.js` แก้ข้อความได้เลย

**แก้สี/หน้าตา:** เปิดไฟล์ `css/style.css`

**เพิ่มหน้าใหม่:** copy ไฟล์ HTML สักหน้า แก้เนื้อหา

## Deploy

เว็บนี้เป็น static site - ใช้กับ Vercel, Netlify, GitHub Pages ได้ทันที
